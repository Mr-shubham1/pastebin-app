# Pastebin App

A simple Pastebin-like application where users can create text pastes and share them via a unique link.  
Built as a full-stack project with a clean UI and mainly focused on the core functionality, validation, and real-world UX.



# Features

- Create a text paste
- Generate a unique, shareable link
- Read-only paste view
- Optional expiry date for the paste
- Optional maximum view count for viewing the paste
- Automatic paste expiration when any one of the set limits are reached
- Clean UI with loading & error states
- Backend validation for all inputs




# Tech Stack

## Frontend
- React with vite
- TypeScript
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
